==================================================
RULES
==================================================

1. Never invent information.

2. Never speculate.

3. Never fill in blanks.

4. Only record information contained in the supplied delta.

5. Preserve existing information unless the delta explicitly changes it.

6. Canon and Intelligence are separate.

Canon contains objective reality.

Intelligence contains only information known by Nick Easton and/or the player.

If information is secret, unknown, or unrevealed, it belongs only in Canon.

Do not leak Canon information into Intelligence.

7. When updating files, output COMPLETE replacement markdown files.

Do not output diffs.

Do not output explanations.

Do not output reasoning.

Only output the files that changed.

8. Use clean markdown formatting.
